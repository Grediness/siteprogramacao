Capitulo3:Codiçoes

Em nossos programas de computador, muitas vezes precisamos tomar decisões. Imagine que você está escrevendo um programa para um semáforo: ele precisa mudar de cor dependendo do tempo que passou. Ou talvez um programa para verificar se um usuário tem permissão para acessar uma determinada área. É aí que entram as condicionais. As condicionais são como o "cérebro decisor" do seu código. Elas permitem que o programa execute diferentes blocos de instruções com base em se uma determinada condição é verdadeira ou falsa. Pense nelas como um "se... então... senão...". Com as condicionais, seu programa ganha a capacidade de se adaptar a diferentes situações, tornando-o muito mais dinâmico e útil. Em Python, as principais ferramentas para implementar essa lógica condicional são as estruturas if, else e elif, que nos permitem definir diferentes caminhos de execução dependendo do resultado de uma avaliação lógica. Elas são a base para criar programas inteligentes que podem responder de maneira variada aos dados e interações do usuário.

if (se):

O if é a estrutura fundamental para verificar se uma condição é verdadeira. Se a condição que você especificar for avaliada como True, o bloco de código indentado abaixo do if será executado. Caso contrário, esse bloco de código será ignorado e o programa seguirá para a próxima instrução após o bloco if.

Imagine que você quer verificar se um número é positivo:

numero = 10

if numero > 0:

print("O número é positivo")

Nesse exemplo, a condição numero > 0 é verdadeira (10 é maior que 0), então a mensagem "O número é positivo" será impressa.

else (senão):

O else é opcional e sempre vem depois de um if (ou uma série de elifs). Ele define um bloco de código que será executado apenas se a condição do if (e de qualquer elif anterior) for avaliada como False. É como dizer: "Se essa condição não for verdadeira, então faça isso aqui."

Vamos adicionar um else ao nosso exemplo para lidar com números não positivos:

numero = -5

if numero > 0:

print("O número é positivo")

else:

print("O número não é positivo")

Agora, como numero > 0 é falso (-5 não é maior que 0), o bloco de código dentro do else será executado, e a mensagem "O número não é positivo" será impressa.

elif (senão se):

O elif (que é uma abreviação de "else if") permite que você verifique múltiplas condições em sequência. Ele vem depois de um if e antes de um else (se houver um). Você pode ter quantos elifs forem necessários. A primeira condição elif que for avaliada como True terá seu bloco de código executado, e as outras condições (elifs e o else) serão ignoradas. Se nenhuma das condições (if ou elifs) for verdadeira, e houver um else, o bloco de código do else será executado.

Vamos criar um exemplo com elif para classificar um número em positivo, negativo ou zero:

numero = 0

if numero > 0:

print("O número é positivo")

elif numero < 0:

print("O número é negativo")

else:

print("O número é zero")

Primeiro, a condição numero > 0 (0 > 0) é falsa.

Então, a condição do elif numero < 0 (0 < 0) também é falsa.

Finalmente, como as condições anteriores foram falsas, o bloco de código dentro do else é executado, e a mensagem "O número é zero" é impressa.

Use if para executar um bloco de código se uma condição for verdadeira.

Use else (opcional) para executar um bloco de código se a condição do if (e de qualquer elif) for falsa.

Use elif (opcional, pode ter vários) para verificar múltiplas condições em sequência.

Se você percebeu os prints que estão nos: if,elif,else, estão um pouco para a direita isso é chamado de identação,isso significa que aquele código está dentro do if, mas poderia ser em vários tipos de funções de várias linguagens.
