FUNÇÃO PRINT

Essa função é usada quando quer mostrar algo na tela. Acredito que quem está lendo isso já fez ou já viu alguém fazendo um pix, se você reparar bem independentemente do valor do pix irá aparecer uma mensagem de: *TRANSFERÊNCIA APROVADA*, isso quer dizer quem a pessoa que desenvolveu esse código em algum momento colocou o código:

print (“*TRANSFERENCIA APROVADA”)*

A função print também serve para mostrar o resultado de alguma conta matemática, sua sintaxe ficaria assim

print (2+3)

Por ser números não há a necessidade de colocar as aspas

OBS:>>>2+3

Esse símbolo significa que há um código antes do 2+3, que nesse caso seria o código: print (2+3).Caso queria fazer outros tipos de conta você irá usar:

VARIAVEL NUMERICA

As mais conhecidas são as variáveis de números inteiros e as variáveis de números ponto flutuante

Números inteiros:1,0,-5,550, -47. (são números sem decimal)

Números ponto flutuante:1.0, 5.478, 10.478, 30000.4(são números decimais)

Em python o ponto(.) é usado para separar a parte inteira da parte decimal ao invés de usar a virgula como é visto no nosso dia a dia.

OBS: existem outros tipos de variáveis numéricas como: números complexos, modulo cmath, ponto fixo, frações.

VARIAVEIS LOGICAS/BOOLENAS

Esses tipos de variáveis após serem usadas irão retornar um valor True ou False. Para usar essas variáveis é necessário usar algo chamado de operadores lógicos.

OPERADOR OPERAÇÃO SIMBOLO MATEMATICO
== Igualdade =
> Maior que >
< Menor que <
!= Diferente ≠
>= Maior ou igual ≥
<= Menor ou Igual ≤

Esses tipos de operadores sempre serão feitos uma comparação, caso a comparação seja verdadeira o resultado será True e caso o resultado for falso será False. Quando foi fazer uma comparação sempre vai ser feito uma pergunta

OPERADOR PYTHON SIGNIFICADO
NOT NÃO
AND E
OR OU

NOT

Esse operador é um operador de negação ou de inversão, isso porque um valor verdadeiro (True) se torna falso e vice-versa

AND

O operador and só irá retornar o valor (True) caso as duas opções sejam verdadeiras, caso o contrário será considerado (False).

OR

E o operador or é bem parecido com o operador and só que a sua única diferença é que para ele retornar o valor True só é preciso que uma das opções seja considerada verdadeira.

STRING

Esse tipo de variação inclui qualquer tipo de caracter(letras,números,sinais de pontuação, etc...). Exemplo:abacaxi. Uma string que apresente mais de um caracter é chamado de cadeia de caracter, para uma melhor explicação uma cadeia de caracteres seria como cada letra de uma palavra(nesse caso)estivesse ocupando um bloco de espaço. Quando você for usar uma variável desse tipo é OBRIGATÓRIO usar aspas tanto no início quanto no final da string. Caso a frase tenha espaços,esse espaço também contará com um caracter. caso você queira saber a quantidade total de caracteres de uma determinada frase ou quer saber em qual posição está um caracter em específico, você irá usar uma função chamada *len()*. Essa função retornar uma valor inteiro que representa a quantidade de caracter.

CÓDIGO

print(len("A"))

TERMINAL:1

CÓDIGO

print(len(O rato roeu a roupa ))

Terminal:19

Como foi dito anteriormente outra possível função de uma string é poder acessar um caracter em específico. Para isso é necessário usar [] e dentro desses colchetes como o número ou como chamamos o índice. A contagem do índice começa no zero. Caso você coloque:

print([-1])

Irá ser mostrado o último caracter.
